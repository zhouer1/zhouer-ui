### 属性

| 属性     | 说明     | 类型    | 默认值  |
| -------- | -------- | ------- | ------- |
| type     | 按钮类型 | `default` \| `dashed` \| `primary` | default |
| size     | 按钮大小 | `default` \| `small` \| `large`  | default |
| disabled | 是否禁用 | boolean | false   |

### 事件

| 事件    | 说明     | 类型                |
| ----- | ------ | ----------------- |
| click | 按钮点击事件 | (e:Event) => void |
